Sortable.js Demo Page

Explore the features of Sortable.js with interactive examples.

1. Simple Sortable List

This example shows a basic sortable list where you can drag and drop items to reorder them.

Item 1
Item 2
Item 3
Item 4
<div class="list" id="simple-list">
  <div class="list-item">Item 1</div>
  <div class="list-item">Item 2</div>
  <div class="list-item">Item 3</div>
  <div class="list-item">Item 4</div>
</div>

new Sortable(document.getElementById('simple-list'), {
  animation: 150
});

2. Drag Between Containers

Here, you can drag items between two different containers. This is achieved by grouping the lists under the same 'group' property.

List 1 - Item 1
List 1 - Item 2
List 1 - Item 3
List 2 - Item 1
List 2 - Item 2
List 2 - Item 3
new Sortable(document.getElementById('list-1'), {
  group: 'shared',
  animation: 150
});

new Sortable(document.getElementById('list-2'), {
  group: 'shared',
  animation: 150
});

4. Drop Zones

This section demonstrates customized drop zones. Drag an item into the drop zone to see the interaction. We define separate groups to manage drop behavior.

Draggable Item 1
Draggable Item 2
Draggable Item 3
Drop items here 
new Sortable(document.getElementById('drag-items'), {
  group: {
    name: 'drop-zone',
    pull: 'clone',
    put: false
  },
  animation: 150
});

new Sortable(document.getElementById('drop-zone-1'), {
  group: {
    name: 'drop-zone',
    pull: false,
    put: true
  },
  animation: 150
});
